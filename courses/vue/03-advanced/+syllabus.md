# Vue.js Advanced Course Syllabus

## Module 9: Performance & Transitions

**Goal**: Optimization and Animation.

1.  **Optimization Techniques**
    - Concepts: `v-memo`, Bundle Size, Lazy Loading Components (`defineAsyncComponent`).
    - Goal: Improve rendering performance.
2.  **Transitions & Animations**
    - Concepts: `<Transition>`, `<TransitionGroup>`, JS Hooks.
    - Goal: Animate entry/leave and lists.

## Module 10: Server-Side & Static

**Goal**: Beyond the SPA.

3.  **Server-Side Rendering (SSR)**
    - Concepts: Nuxt.js overview, hydration.
    - Goal: SEO-friendly Vue applications.
4.  **Static Site Generation (SSG)**
    - Concepts: VitePress / Nuxt Generate.
    - Goal: Build documentation or blogs.

## Module 11: Data & Networking

**Goal**: Robust Data Handling.

5.  **Advanced Data Fetching**
    - Concepts: TanStack Query (Vue Query) or tailored Composables.
    - Goal: Caching, Deduping, and Optimistic UI.
6.  **GraphQL Integration**
    - Concepts: Apollo Client with Vue.
    - Goal: Querying GraphQL endpoints.

## Module 12: Testing & Deployment

**Goal**: Quality Assurance.

7.  **Unit Testing**
    - Concepts: Vitest, Vue Test Utils.
    - Goal: Test component logic and rendering.
8.  **End-to-End (E2E)**
    - Concepts: Cypress or Playwright.
    - Goal: Test user flows.
9.  **Deployment**
    - Concepts: Building for production, Dockerizing (nginx).
    - Goal: Ship the app.
