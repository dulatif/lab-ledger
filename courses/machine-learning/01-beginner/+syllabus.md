# Machine Learning Beginner Course Syllabus

## Module 1: Mathematical Foundations

**Goal**: The math you strictly need (no fluff).

1.  **Linear Algebra**
    - Concepts: Vectors, Matrices, Dot Product, Eigenvalues.
    - Goal: Understand how data is represented as tensors.
2.  **Calculus**
    - Concepts: Derivatives, Chain Rule, Gradients.
    - Goal: Understand "Rate of Change" for optimization.
3.  **Probability & Statistics**
    - Concepts: Distributions (Normal, Bernoulli), Mean, Variance, Bayes Theorem.
    - Goal: Quantify uncertainty.

## Module 2: Python for Machine Learning

**Goal**: The toolset.

4.  **NumPy & Vectorization**
    - Concepts: Arrays over Lists, Broadcasting, Performance.
    - Goal: Fast math operations.
5.  **Pandas**
    - Concepts: DataFrames, Cleaning, Manipulation.
    - Goal: Handling tabular data (Excel/CSV).
6.  **Visualization (Matplotlib/Seaborn)**
    - Concepts: Plots, Histograms, Heatmaps.
    - Goal: "Seeing" the data distribution.

## Module 3: Data Engineering Pipeline

**Goal**: Garbage In, Garbage Out.

7.  **Data Collection & Formats**
    - Concepts: CSV, JSON, Parquet, SQL.
    - Goal: Loading data.
8.  **Preprocessing & Cleaning**
    - Concepts: Missing values (Imputation), Outliers.
    - Goal: Making data valid.
9.  **Feature Engineering**
    - Concepts: Scaling (MinMax/Standard), Encoding (One-Hot).
    - Goal: Making data distinct.
