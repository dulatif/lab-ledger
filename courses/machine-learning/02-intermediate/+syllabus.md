# Machine Learning Intermediate Course Syllabus

## Module 4: Supervised Learning

**Goal**: Predicting labels.

1.  **Workflow & Scikit-Learn**
    - Concepts: Train/Test Split, Fit/Predict API.
    - Goal: The standard ML loop.
2.  **Regression**
    - Concepts: Linear Regression, Cost Function (MSE), Gradient Descent.
    - Goal: Predicting prices/numbers.
3.  **Classification**
    - Concepts: Logistic Regression, KNN, Decision Trees, SVM.
    - Goal: Predicting categories (spam/ham).

## Module 5: Unsupervised Learning

**Goal**: Finding hidden structures.

4.  **Clustering**
    - Concepts: K-Means, Hierarchical Clustering.
    - Goal: Customer segmentation.
5.  **Dimensionality Reduction**
    - Concepts: PCA (Principal Component Analysis).
    - Goal: Compressing data while keeping information.

## Module 6: Model Evaluation & Tuning

**Goal**: Improving performance.

6.  **Metrics**
    - Concepts: Accuracy, Precision, Recall, F1-Score, ROC-AUC.
    - Goal: Knowing if the model is actually good.
7.  **Overfitting vs Underfitting**
    - Concepts: Bias-Variance Tradeoff, Regularization (L1/L2).
    - Goal: Generalization.
8.  **Validation Strategies**
    - Concepts: Cross-Validation (K-Fold).
    - Goal: Robust testing.
