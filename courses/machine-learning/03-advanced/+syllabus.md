# Machine Learning Advanced Course Syllabus

## Module 7: Deep Learning Foundations

**Goal**: Neural Networks.

1.  **The Perceptron**
    - Concepts: Neurons, Weights, Biases, Activation Functions (ReLU, Sigmoid).
    - Goal: The building block of DL.
2.  **Backpropagation**
    - Concepts: Loss Functions, Optimizers (Adam, SGD).
    - Goal: How networks learn.
3.  **Frameworks (PyTorch/TensorFlow)**
    - Concepts: Tensors, Autograd, Model Subclassing.
    - Goal: Building a network in code.

## Module 8: Specialized Architectures

**Goal**: Vision and Sequence.

4.  **Computer Vision (CNN)**
    - Concepts: Convolution, Pooling, Strides.
    - Goal: Image classification.
5.  **Sequence Models (RNN/LSTM)**
    - Concepts: Memory, Time-series.
    - Goal: Text/Stock prediction.
6.  **Reinforcement Learning**
    - Concepts: Agents, Rewards, Policies.
    - Goal: Learning by interaction.

## Module 9: Modern AI & NLP

**Goal**: State of the Art.

7.  **Attention & Transformers**
    - Concepts: "Attention is All You Need", Self-Attention.
    - Goal: Understanding BERT/GPT architecture.
8.  **NLP Pipeline**
    - Concepts: Tokenization, Embeddings (Word2Vec), Stemming.
    - Goal: Processing text.
9.  **Generative AI**
    - Concepts: GANs, Autoencoders.
    - Goal: Creating new data.
