# MongoDB Beginner Syllabus

## Module 1: Introduction & Setup

**Goal**: Understand NoSQL and set up the environment.

1.  **SQL vs NoSQL**
    - Concepts: Relational vs Document Model. JSON/BSON.
    - Goal: Why use MongoDB?
2.  **Environment Setup**
    - Concepts: MongoDB Atlas (Cloud), Compass (GUI), Mongosh (Shell).
    - Goal: Connect to a database.

## Module 2: CRUD Basics

**Goal**: Create, Read, Update, Delete.

3.  **Inserting Documents**
    - Concepts: `insertOne`, `insertMany`, `_id`.
    - Goal: Add data.
4.  **Finding Documents**
    - Concepts: `find`, Query Operators ($eq, $gt, $in).
    - Goal: Query data.
5.  **Updating & Deleting**
    - Concepts: `updateOne`, `updateMany`, $set, `deleteOne`.
    - Goal: Modify data.

## Module 3: Data Modeling

**Goal**: Designing schemas.

6.  **Embedded vs References**
    - Concepts: 1:1, 1:Many, Many:Many relationships.
    - Goal: When to embed?
7.  **Schema Design Patterns**
    - Concepts: Bucket pattern, Polymorphic pattern.
    - Goal: Efficient storage.
