# Lesson 2: Setup

## Learning Goals

- Atlas.
- Compass.

## 1. MongoDB Atlas

The fully managed cloud service (AWS/GCP/Azure).
Free Tier (M0 Sandbox) is perfect for learning.

## 2. MongoDB Compass

The official GUI.
Allows you to explore schema, run queries visually, and explain plans.

## 3. Mongosh

The JavaScript-based shell.
`db.users.find()`

## 4. Connection String

`mongodb+srv://<user>:<password>@cluster0.mongodb.net/test`
Keep this secret!

## Key Takeaways

- Use Atlas for easiest setup.
- Use Compass for finding data/debugging.
