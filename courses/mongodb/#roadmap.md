# MongoDB Developer Roadmap

## Beginner

- **Introduction to NoSQL**: SQL vs NoSQL, Documents, Collections.
- **CRUD Operations**: Insert, Find, Update, Delete.
- **Data Modeling**: Embedded Docs vs References (Normalization vs Denormalization).
- **Compass & Atlas**: Using the GUI and Cloud.

## Intermediate

- **Aggregation Framework**: Pipelines, $match, $group, $project.
- **Indexing**: Single Field, Compound, Multikey, Winning Plans.
- **Transactions**: ACID in MongoDB.
- **Schema Validation**: Enforcing rules.

## Advanced

- **Replication**: Replica Sets, Oplog, High Availability.
- **Sharding**: Horizontal Scaling, Shard Keys.
- **Performance Tuning**: Profiling, Explain Plans.
- **Security**: Authentication, Authorization, Encryption.
