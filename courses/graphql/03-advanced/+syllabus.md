# GraphQL Advanced Course Syllabus

## Module 8: Real-Time GraphQL

**Goal**: Push data to clients.

1.  **Subscriptions**
    - Concepts: Pub/Sub systems.
    - Goal: Build a chat app feature.
2.  **WebSockets vs SSE**
    - Concepts: Transport protocols for keep-alive connections.
    - Goal: Choose the right transport.
3.  **Live Queries & Defer**
    - Concepts: `@defer`, `@stream`.
    - Goal: Optimize large query delivery.

## Module 9: Performance & Best Practices

**Goal**: Scale safely.

4.  **Pagination**
    - Concepts: Limit/Offset vs Cursor-based (Relay Style).
    - Goal: Implement infinite scroll.
5.  **Security & Complexity**
    - Concepts: Query Depth Limit, Rate Limiting, Complexity Analysis.
    - Goal: Prevent DOS attacks via expensive queries.
6.  **Schema Federation**
    - Concepts: Microservices, Apollo Federation.
    - Goal: Stitch multiple graphs into one.

## Module 10: Implementations & Ecosystem

**Goal**: Survey the landscape.

7.  **Server Landscape**
    - Concepts: Apollo Server, GraphQL Yoga, Mercurius.
    - Goal: Choosing the right engine.
8.  **Client Landscape**
    - Concepts: Apollo Client, Relay, Urql.
    - Goal: Choosing the right client.
