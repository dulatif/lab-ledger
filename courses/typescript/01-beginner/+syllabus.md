# TypeScript Beginner Course Syllabus

## Module 1: Ecosystem & Configuration

**Goal**: Setup the environment.

1.  **Introduction & Compilation**
    - Concepts: TS vs JS, `tsc`.
    - Goal: Run your first TS file.
2.  **Configuration (tsconfig.json)**
    - Concepts: `strict`, `target`, `outDir`.
    - Goal: Customize the compiler.

## Module 2: The Type System (Primitives)

**Goal**: Using types.

3.  **Primitives & Arrays**
    - Concepts: `string`, `number`, `boolean`, `[]`.
    - Goal: Typing variables.
4.  **Special Types**
    - Concepts: `any`, `unknown`, `void`, `never`.
    - Goal: Handling edge cases.
5.  **Enums & Tuples**
    - Concepts: `enum`, `[string, number]`.
    - Goal: Structures not in JS.

## Module 3: Functions & Objects Basics

**Goal**: Typing logic.

6.  **Typing Objects**
    - Concepts: `type` alias, `interface` (basics).
    - Goal: Defining shapes.
7.  **Typing Functions**
    - Concepts: Parameter types, Return types, Inference.
    - Goal: Safe functions.
