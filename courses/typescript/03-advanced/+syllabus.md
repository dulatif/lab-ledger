# Advanced TypeScript Course Syllabus

Based on Modules 7, 8, 9, and 10 of the roadmap.

## Module 7: Generics

1. Introduction to Generics
2. Generic Constraints
3. Validation with `satisfies` operator

## Module 8: Utility Types

4. Transformation Types (`Partial`, `Pick`, `Omit`, `Record`)
5. Set Operation Types (`Exclude`, `Extract`)
6. Function Helper Types (`Parameters`, `ReturnType`, `InstanceType`)
7. Async Handling with `Awaited`
8. Nullability with `NonNullable`

## Module 9: Advanced Type Manipulation

9. Conditional Types
10. Mapped Types
11. Literal Types and Template Literal Types
12. Recursive Types
13. Decorators for Metaprogramming

## Module 10: Modules & Ecosystem

14. TypeScript Module System and External Modules
15. Namespaces and Augmentation
16. Tooling (Linting, Formatting, Build Tools)
17. Useful Packages in Node.js Ecosystem
