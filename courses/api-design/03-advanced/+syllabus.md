# Advanced API Design Course Syllabus

Based on Modules 7, 8, and 9 of the roadmap.

## Module 7: Advanced API Architectures

1. GraphQL vs REST vs gRPC
2. Sync vs Async APIs (Webhooks vs Polling)
3. Microservices and API Gateways
4. Event-Driven Architecture (RabbitMQ/Kafka)

## Module 8: Performance Engineering

5. Caching Strategies (Client, Server, CDN)
6. Rate Limiting and Throttling Algorithms
7. Load Balancing Strategies
8. Reliability Patterns (Circuit Breaker, Retries)
9. Observability: Profiling and Metrics

## Module 9: Real-Time, Compliance & Governance

10. Real-Time Data (WebSockets & SSE)
11. Contract Testing in Distributed Systems
12. API Lifecycle Management & Governance
13. Legal Compliance (GDPR, PCI DSS, PII)
