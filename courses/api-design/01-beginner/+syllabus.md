# Beginner API Design Course Syllabus

Based on Modules 1, 2, and 3 of the roadmap.

## Module 1: Network Fundamentals

1. Web Standards and API Basics
2. Infrastructure: TCP/IP and DNS Basics
3. Anatomy of an HTTP Request (URL, Query, Path, Body)

## Module 2: The HTTP Protocol

4. HTTP Protocol Basics and Versions (1.1, 2, 3)
5. HTTP Methods Deep Dive (GET, POST, PUT, DELETE, PATCH)
6. Status Codes and What They Mean
7. Headers and Content Negotiation
8. State Management (Cookies & Caching)
9. Browser Security (CORS)

## Module 3: Introduction to API Styles

10. Overview of API Styles (REST, SOAP, GraphQL, RPC)
11. Introduction to RESTful APIs and JSON
12. Legacy Systems: A Brief Look at SOAP
