# Intermediate API Design Course Syllabus

Based on Modules 4, 5, and 6 of the roadmap.

## Module 4: RESTful API Design

1. URI Design and Resource Modeling
2. Handling CRUD Operations (Methods & Status Codes)
3. Pagination, Filtering, and Sorting
4. Versioning Strategies
5. Idempotency and Reliability
6. Advanced Standards: RFC 7807 and HATEOAS

## Module 5: Security & Authentication

7. Authentication vs Authorization
8. API Keys and Basic Auth
9. Modern Auth: JWT and OAuth 2.0
10. Access Control (RBAC vs ABAC)
11. Security Best Practices (CORS, CSP, Rate Limiting)

## Module 6: Documentation & Testing

12. API Documentation Standards (OpenAPI/Swagger)
13. Testing Tools (Postman)
14. Testing Strategies (Unit, Integration, Functional)
15. Mocking APIs for Parallel Development
