# React Native Intermediate Course Syllabus

## Module 4: Platform Integration

**Goal**: Handle platform differences and accessibility.

1.  **Platform Specific Code**
    - Concepts: `Platform.OS`, `Platform.select`, `.ios.js`/`.android.js` extensions.
    - Goal: Customize UI for iOS (Shadows) vs Android (Elevation).
2.  **React Native Web**
    - Concepts: Running the same code in a browser.
    - Goal: Understand the "Write Once, Run Everywhere" promise.
3.  **Accessibility**
    - Concepts: `accessibilityLabel`, `accessibilityRole`, Screen Reader testing.
    - Goal: Make the app usable for everyone.

## Module 5: Interactions & Navigation

**Goal**: Create a multi-screen app with smooth interactions.

4.  **Navigation (React Navigation)**
    - Concepts: Stack Navigator, Tab Navigator, Drawer Navigator.
    - Goal: Implement a full app navigation flow.
5.  **Advanced Navigation**
    - Concepts: Passing params between screens, Deep Linking config.
    - Goal: Handle "Open in App" links from the web.
6.  **Animations**
    - Concepts: `Animated` API, LayoutAnimation, Reanimated (Intro).
    - Goal: Add micro-interactions (fade-ins, slide-ups).

## Module 6: Data & Networking

**Goal**: Connect the app to the real world.

7.  **Networking (Fetch/Axios)**
    - Concepts: REST API calls, Handling Loading/Error states, `NetInfo`.
    - Goal: Fetch and display data from a public API.
8.  **Local Storage**
    - Concepts: `AsyncStorage`, SecureStore (for tokens).
    - Goal: Persist user login session and preferences.
9.  **Real-time Updates**
    - Concepts: WebSockets.
    - Goal: Build a simple chat feature.
