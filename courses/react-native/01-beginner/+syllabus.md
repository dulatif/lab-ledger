# React Native Beginner Course Syllabus

## Module 1: Prerequisites & Setup

**Goal**: Set up the environment and understand the core React Native workflow.

1.  **Web Foundations Refresher**
    - Concepts: JavaScript ES6 features (Arrow functions, Destructuring), CSS Flexbox basics.
    - Goal: Ensure readiness for React Native syntax.
2.  **React Core Concepts**
    - Concepts: JSX, Components (Functional), State (`useState`), Props.
    - Goal: Master the building blocks of React.
3.  **Environment Setup**
    - Concepts: Expo vs React Native CLI, Metro Bundler, Android Studio/Xcode basics.
    - Goal: Run "Hello World" on a physical device or emulator.

## Module 2: Core Components & Layout

**Goal**: Build static user interfaces using native components.

4.  **Basic Views & Usage**
    - Concepts: `View` (Div), `SafeAreaView` (Notch handling), `Text`, `StyleSheet`.
    - Goal: Implement a basic static layout.
5.  **Handling User Input**
    - Concepts: `TextInput`, `Button` vs `Pressable`, Keyboard handling (`KeyboardAvoidingView`).
    - Goal: Create a login form.
6.  **Media & Controls**
    - Concepts: `Image` (Network/Local), `Switch`, `ActivityIndicator`, `Modal`.
    - Goal: Add visual richness to the app.

## Module 3: Styling & Lists

**Goal**: Handle dynamic content and professional styling.

7.  **Styling & Flexbox**
    - Concepts: Flex Direction, Justify Content, Align Items, Absolute vs Relative positioning.
    - Goal: Replicate a complex UI design.
8.  **Scrollable Lists**
    - Concepts: `ScrollView` vs `FlatList` (Performance).
    - Goal: Render a list of 100+ items efficiently.
9.  **SectionList & Gestures**
    - Concepts: `SectionList` (Grouped data), `RefreshControl` (Pull to refresh).
    - Goal: Build a contacts list with headers.
