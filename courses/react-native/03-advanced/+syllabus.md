# React Native Advanced Course Syllabus

## Module 7: Security & Native Modules

**Goal**: Secure the app and bridge the gap to native code.

1.  **App Security**
    - Concepts: SSL Pinning, Secure Storage KeyChain/Keystore, Biometrics.
    - Goal: Secure sensitive user data.
2.  **Native Modules**
    - Concepts: Bridging Swift/Kotlin code to JavaScript.
    - Goal: Access a native API not available in standard React Native.
3.  **Push Notifications**
    - Concepts: Expo Notifications / Firebase Cloud Messaging (FCM).
    - Goal: Send specific alerts to users.

## Module 8: Testing Strategy

**Goal**: Ensure app reliability via automated testing.

4.  **Unit & Component Testing**
    - Concepts: Jest, React Native Testing Library.
    - Goal: Test pure functions and component rendering.
5.  **End-to-End (E2E) Testing**
    - Concepts: Detox / Maestro / Appium.
    - Goal: Automate a full login -> checkout flow on an emulator.

## Module 9: Performance & Deployment

**Goal**: Optimize for 60fps and publish to stores.

6.  **Performance Optimization**
    - Concepts: `memo`, `useCallback`, FlatList optimization (`getItemLayout`), Hermes Engine.
    - Goal: Debug and fix JS thread blocks.
7.  **Profiling**
    - Concepts: Flipper / React DevTools Profiler, FPS monitoring.
    - Goal: Identify re-render issues.
8.  **Deployment**
    - Concepts: CD/CI (EAS Build / Fastlane), App Store Connect, Google Play Console.
    - Goal: Prepare a release build for app stores.
