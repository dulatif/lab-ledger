# Advanced SQL Course Syllabus

Based on Modules 8, 9, and 10 of the roadmap.

## Module 8: Performance Optimization

1. Introduction to Indexing
2. Query Analysis and Explain Plans
3. Optimization Strategies (Joins vs Subqueries)

## Module 9: Transactions & Security

4. ACID Properties and Transactions
5. Controlling Transactions (COMMIT, ROLLBACK)
6. Security and Permissions (GRANT, REVOKE)

## Module 10: Advanced Features

7. Window Functions (RANK, LEAD/LAG)
8. CTEs and Recursive Queries
9. Stored Procedures and Functions
10. Pivot and Unpivot Operations
