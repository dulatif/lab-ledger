# Intermediate SQL Course Syllabus

Based on Modules 4, 5, 6, and 7 of the roadmap.

## Module 4: Aggregation & Grouping

1. Aggregate Functions (SUM, COUNT, AVG, MAX/MIN)
2. Grouping Results (GROUP BY, HAVING)

## Module 5: Joining Tables

3. Understanding Joins (INNER, LEFT, RIGHT, FULL)
4. Advanced Joins (Self Join, Cross Join)

## Module 6: Advanced Functions

5. String Manipulation Functions (CONCAT, SUBSTRING, etc.)
6. Numeric and Math Functions
7. Date and Time Functions
8. Conditional Logic (CASE, COALESCE)

## Module 7: Subqueries & Views

9. Writing Nested Subqueries
10. Working with Views (Create, Modify, Drop)
