**RDBMS (Relational Database Management System)**:
Data is stored in **Tables** (Rows & Columns). Tables are related via Keys.
Examples: PostgreSQL, MySQL, SQL Server.

- **Structured**: Schema must be defined upfront.
- **ACID**: Guarantees reliability.

**NoSQL**:
Document, Key-Value, or Graph. Flexible schema.
Examples: MongoDB, Redis.
