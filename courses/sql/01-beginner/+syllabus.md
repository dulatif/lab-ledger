# Beginner SQL Course Syllabus

Based on Modules 1, 2, and 3 of the roadmap.

## Module 1: Relational Concepts

1. What are Relational Databases? (RDBMS vs NoSQL)
2. Core SQL Syntax and Data Types

## Module 2: Defining Structure (DDL)

3. Creating and Dropping Tables (CREATE, DROP)
4. Altering Tables (ALTER, TRUNCATE)
5. Ensuring Data Integrity with Constraints (PK, FK, UNIQUE)

## Module 3: Manipulating Data (DML)

6. Adding Data (INSERT)
7. Modifying and Deleting Data (UPDATE, DELETE)
8. Basic Queries (SELECT, FROM, WHERE)
9. Sorting and Filtering (ORDER BY, DISTINCT)
