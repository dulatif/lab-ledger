# Flutter Advanced Course Syllabus

## Module 7: Animations & Internals

**Goal**: Make it smooth 60fps.

1.  **Implicit Animations**
    - Concepts: `AnimatedContainer`, `AnimatedOpacity`.
    - Goal: Simple "fire and forget" animations.
2.  **Explicit Animations**
    - Concepts: `AnimationController`, `Tween`, `CurvedAnimation`.
    - Goal: Complex, controlled animations.
3.  **Flutter Internals**
    - Concepts: The 3 Trees (Widget, Element, RenderObject).
    - Goal: Understand how Flutter draws pixels.
4.  **Isolates (Concurrency)**
    - Concepts: `Isolate.spawn` vs `compute`.
    - Goal: Run heavy math off the UI thread.

## Module 8: Quality Assurance & DevTools

**Goal**: Ship with confidence.

5.  **Testing Strategy**
    - Concepts: Unit Tests (Dart), Widget Tests (PumpWidget), Integration Tests.
    - Goal: Write a comprehensive test suite.
6.  **Debugging & Profiling**
    - Concepts: Flutter Inspector, DevTools (Memory/Performance views).
    - Goal: Fix memory leaks and jank.

## Module 9: DevOps & Deployment

**Goal**: Release to the world.

7.  **Analytics & Crashlytics**
    - Concepts: Monitoring app health.
    - Goal: Understand user behavior and crashes.
8.  **CI/CD**
    - Concepts: GitHub Actions / Codemagic.
    - Goal: Automate builds.
9.  **Publishing**
    - Concepts: Signing (Keystore), App Store Connect, Google Play Console.
    - Goal: Publish the app.
