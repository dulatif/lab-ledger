# Flutter Intermediate Course Syllabus

## Module 4: Networking & Storage

**Goal**: Connect to the world and save data.

1.  **Networking (HTTP)**
    - Concepts: `http` package, REST APIs, `async`/`await`.
    - Goal: Fetch JSON data from an API.
2.  **JSON & Serialization**
    - Concepts: `dart:convert`, JSON encoding/decoding, Model classes.
    - Goal: Convert API responses to Dart Objects.
3.  **Local Storage**
    - Concepts: `shared_preferences` (KeyValue), `sqlite` (Relational).
    - Goal: Persist user settings.
4.  **Cloud Integration (Firebase)**
    - Concepts: Firebase Auth, Firestore.
    - Goal: Basic User Authentication.

## Module 5: State Management

**Goal**: Manage complex app states.

5.  **InheritedWidget & Provider**
    - Concepts: Lifting State Up, `Provider` package basics.
    - Goal: Share state across the app.
6.  **Riverpod / BLoC**
    - Concepts: Modern state management. Streams vs Notifiers.
    - Goal: Implement a scalable state pattern.
7.  **Reactivity**
    - Concepts: `ValueNotifier`, `ChangeNotifier`.
    - Goal: React to data changes efficiently.

## Module 6: Advanced Dart & Design

**Goal**: Write professional, clean code.

8.  **Asynchronous Dart**
    - Concepts: `Future`, `Stream`, `StreamBuilder`.
    - Goal: Handle real-time data flow.
9.  **Null Safety & OOP**
    - Concepts: Sound Null Safety, Mixins, Abstract Classes, Interfaces.
    - Goal: Write robust, crash-free code.
10. **Functional Concepts**
    - Concepts: List methods (`map`, `filter`, `reduce`), Lambdas.
    - Goal: Write declarative transformations.
