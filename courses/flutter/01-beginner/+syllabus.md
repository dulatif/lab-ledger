# Flutter Beginner Course Syllabus

## Module 1: Dart Fundamentals & Setup

**Goal**: Master the language and set up your dev environment.

1.  **Environment Setup**
    - Concepts: Flutter CLI (`flutter doctor`), Android Studio / Xcode.
    - Goal: Run the default counter app on an Emulator.
2.  **Dart Basics (Part 1)**
    - Concepts: `main()` function, Variables (`var`, `final`, `const`), Built-in Types.
    - Goal: Write simple Dart scripts.
3.  **Dart Basics (Part 2)**
    - Concepts: Functions, Operators, Control Flow (If/Switch/Loops).
    - Goal: Understand logic flow in Dart.

## Module 2: The Widget System

**Goal**: Learn that "Everything is a Widget".

4.  **Widgets 101**
    - Concepts: `StatelessWidget` vs `StatefulWidget`. `build()` method.
    - Goal: Create a custom widget.
5.  **Material & Cupertino**
    - Concepts: `Scaffold`, `AppBar`, `Text`, `Button`.
    - Goal: Build a standard screen layout.
6.  **Assets & Resources**
    - Concepts: `pubspec.yaml`, `AssetImage`, Custom Fonts.
    - Goal: Add images and fonts to the app.

## Module 3: Layout & Version Control

**Goal**: Arrange widgets on the screen.

7.  **The Box Model & Flex**
    - Concepts: `Container`, `Padding`, `Margin`, `Row`, `Column`.
    - Goal: Master Flexbox-like layout in Flutter.
8.  **Responsive Layouts**
    - Concepts: `Expanded`, `Flexible`, `Stack`, `MediaQuery`.
    - Goal: Build layouts that adapt to screen size.
9.  **Git for Flutter**
    - Concepts: `.gitignore` for Flutter, Version Control best practices.
    - Goal: Commit your project safely.
