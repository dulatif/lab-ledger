# Data Structures & Algorithms Intermediate Course Syllabus

## Module 4: Sorting Algorithms

**Goal**: Master efficient sorting techniques.

1.  **Elementary Sorts**
    - Concepts: Bubble Sort, Insertion Sort, Selection Sort.
    - Goal: Understand brute force sorting (`O(n^2)`).
2.  **Efficient Sorts: Divide & Conquer**
    - Concepts: Merge Sort, Quick Sort.
    - Goal: Implement `O(n log n)` algorithms and recursion.
3.  **Heap Sort**
    - Concepts: Binary Heap properties, Heapify, Sort.
    - Goal: Understand in-place `O(n log n)` sorting.

## Module 5: Tree Data Structures

**Goal**: Implement hierarchical structures.

4.  **Binary Trees & BST**
    - Concepts: Root, Child, Leaf, Height, BST Properties.
    - Goal: Implement Insertion, Search, and Deletion in a BST.
5.  **Tree Traversals**
    - Concepts: DFS (In-Order, Pre-Order, Post-Order), BFS (Level Order).
    - Goal: Navigate tree structures effectively.
6.  **Heaps (Priority Queues)**
    - Concepts: Min-Heap, Max-Heap, Priority Queue operations.
    - Goal: Efficiently find the min/max element.

## Module 6: Graph Data Structures

**Goal**: Model relationships and networks.

7.  **Graph Representations**
    - Concepts: Adjacency Matrix vs Adjacency List.
    - Goal: Store Directed and Undirected graphs.
8.  **Graph Traversals**
    - Concepts: Breadth-First Search (BFS), Depth-First Search (DFS).
    - Goal: Systematically visit nodes in a graph.
9.  **Shortest Path Algorithms**
    - Concepts: Dijkstra (Weighted), Bellman-Ford (Negative weights), A\* (Heuristic).
    - Goal: Find the optimal path between nodes.
