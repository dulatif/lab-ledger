# Data Structures & Algorithms Advanced Course Syllabus

## Module 7: Advanced & Complex Structures

**Goal**: Solve specialized problems using advanced indexing and trees.

1.  **Self-Balancing Trees**
    - Concepts: AVL Trees (Rotations), B-Trees (Databases).
    - Goal: Maintain `O(log n)` performance dynamically.
2.  **Range Queries**
    - Concepts: Segment Trees, Fenwick Trees (Binary Indexed Trees).
    - Goal: Efficiently query ranges and update values.
3.  **String Structures (Tries)**
    - Concepts: Trie (Prefix Tree), Suffix Trees.
    - Goal: Optimize autocomplete and string searching.
4.  **Disjoint Set (Union-Find)**
    - Concepts: Find, Union, Path Compression.
    - Goal: Manage connected components in `O(alpha(n))`.

## Module 8: Graph Optimization

**Goal**: Solve complex network problems.

5.  **Minimum Spanning Trees (MST)**
    - Concepts: Prim's Algorithm (Greedy Node), Kruskal's Algorithm (Greedy Edge).
    - Goal: Connect a graph with minimum total weight.

## Module 9: Problem Solving Techniques

**Goal**: Master competitive programming paradigms.

6.  **Algorithmic Paradigms**
    - Concepts: Brute Force, Backtracking (N-Queens), Greedy (Huffman), Divide & Conquer.
    - Goal: Identify the right strategy for a problem.
7.  **Dynamic Programming (DP)**
    - Concepts: Memoization (Top-Down), Tabulation (Bottom-Up), Knapsack Problem.
    - Goal: Optimize recursive problems by storing states.
8.  **Advanced Pointers & Patterns**
    - Concepts: Two Pointers, Sliding Window, Fast/Slow Pointers.
    - Goal: Optimize `O(n^2)` array problems to `O(n)`.
