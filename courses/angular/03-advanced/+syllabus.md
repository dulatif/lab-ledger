# Angular Advanced Course Syllabus

## Module 7: Modern State & Signals

**Goal**: The future of Angular reactivity.

1.  **Signals Architecture**
    - Concepts: `signal()`, `computed()`, `effect()`.
    - Goal: Implement fine-grained reactivity.
2.  **Global State (NGRX)**
    - Concepts: Store, Actions, Reducers, Selectors.
    - Goal: Manage complex app state.
3.  **Performance Optimization**
    - Concepts: OnPush Change Detection, TrackBy (or `@for` tracking).
    - Goal: Minimize render cycles.

## Module 8: Advanced Features

**Goal**: Professional-grade patterns.

4.  **Guards & Lazy Loading**
    - Concepts: `CanActivate`, `loadChildren`, Standalone Routing.
    - Goal: Secure routes and speed up initial load.
5.  **Directives & Pipes Deep Dive**
    - Concepts: Creating Custom Directives/Pipes.
    - Goal: Extend HTML behavior.
6.  **Content Projection**
    - Concepts: `<ng-content>`, Multi-slot projection.
    - Goal: Build flexible layout components (Tabs, Cards).

## Module 9: Testing & Deployment

**Goal**: Reliability and Operations.

7.  **Unit Testing**
    - Concepts: Karma/Jasmine (or Jest), `TestBed`.
    - Goal: Test components and services.
8.  **Deployment**
    - Concepts: Build Optimization (`ng build`), Environments.
    - Goal: Deploy to Netlify/Vercel/Firebase.
9.  **SSR & Hydration**
    - Concepts: Angular Universal / SSR with Hydration.
    - Goal: Improve SEO and First Contentful Paint.
