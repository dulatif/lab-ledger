# Angular Beginner Course Syllabus

## Module 1: Foundations & Architecture

**Goal**: Understand the framework's architecture and setup.

1.  **Angular Essentials**
    - Concepts: Architecture, History, TypeScript refresher.
    - Goal: Understand "Why Angular?".
2.  **Project Setup**
    - Concepts: Angular CLI (`ng new`), File Structure (`angular.json`).
    - Goal: Create and run a new project.
3.  **Module System**
    - Concepts: NgModules (`@NgModule`), Feature Modules, declarations/imports.
    - Goal: Understand how Angular organizes code.

## Module 2: Components & Templates

**Goal**: Master the core building block of Angular.

4.  **Component Anatomy**
    - Concepts: Metadata (`@Component`), Selector, Template, Styles.
    - Goal: Create a custom component.
5.  **Data Binding**
    - Concepts: Interpolation `{{}}`, Property Binding `[]`, Event Binding `()`.
    - Goal: Display dynamic data and handle clicks.
6.  **Directives**
    - Concepts: Structural (`*ngIf`, `@if`), Attribute (`[ngClass]`).
    - Goal: Conditionally render content.

## Module 3: Component Communication & Modern Syntax

**Goal**: Handle data flow and modern features.

7.  **Input & Output**
    - Concepts: passing data down (`@Input`), emitting events up (`@Output`).
    - Goal: Build a re-usable card component.
8.  **Control Flow (New Syntax)**
    - Concepts: `@if`, `@for`, `@switch` (Angular 17+).
    - Goal: Render lists and conditions efficiently.
9.  **Standard Pipes**
    - Concepts: `date`, `currency`, `json`.
    - Goal: Format data in the template.
